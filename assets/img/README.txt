  KLINIK — Guide d'ajout des images
═══════════════════════════════════════════════

Placez vos images dans ce dossier (assets/img/) avec les noms exacts suivants :

HERO (slideshow — 4 images qui défilent toutes les 6 secondes)
──────────────────────────────────────────────
  hero1.jpg   ← 1ère image
  hero2.jpg   ← 2ème image
  hero3.jpg   ← 3ème image
  hero4.jpg   ← 4ème image
  Format recommandé : JPG ou JFIF renommé en .jpg
  Dimensions recommandées : 1920 x 1080 px minimum

ABOUT (section "À propos")
──────────────────────────────────────────────
  about.jpg
  Dimensions recommandées : 800 x 600 px

LOGO (navbar + footer)
──────────────────────────────────────────────
  logo.png    ← fond transparent recommandé
  Hauteur recommandée : 100 px (largeur libre)
